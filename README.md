 Developed a robust backend for a voting application using Node.js, Express.js, MongoDB, and JWT tokens, designed for secure
 voting with distinct user and admin functionalities. The application ensures real-time vote count updates and dynamically sorts
 candidate lists based on votes, enhancing transparency and engagement.
 Key Features:
 Advanced Authentication & Authorization: Secure JWT-based authentication for users and admins.
 Admin Dashboard: Manage candidates: add, update, and delete.
 User Voting Experience: Secure voting for authenticated users.
 Dynamic Candidate Ranking: Automatically sorts candidates by vote count in real-time.
 Robust Database Management: Scalable MongoDB integration for data storage.
 Enhanced Security Measures: Password hashing with bcrypt to securely store user passwords.
